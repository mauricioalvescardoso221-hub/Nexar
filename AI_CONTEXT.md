# AI PROJECT CONTEXT – Maurício Eletricista Landing Page

## Project Goal
This is a high-conversion single-page landing website built for Google Ads traffic.
The main goal is lead generation through:
- Click-to-call
- WhatsApp messages
- Contact form submissions

The page must be fast, simple, and optimized for conversion.

## Conversion Strategy

- Strong CTA above the fold
- Phone number visible immediately
- WhatsApp button visible and repeated
- CTA repeated at least 3 times
- Urgency-driven messaging
- Minimal distractions
- No unnecessary sections

---

## Design Guidelines

- Mobile-first design
- Clean and professional look
- Primary color: Dark Blue
- Accent color: Yellow
- High contrast CTA buttons
- Clear typography
- Proper spacing using Tailwind spacing scale
- Sticky WhatsApp floating button on mobile

No heavy animations.
No complex UI elements.
Focus on clarity and trust.

---

## SEO Requirements

- One H1 only
- H1 must include: "Electrician in [City Name]"
- Proper semantic HTML structure
- Meta title and description
- Local SEO optimized copy
- Accessible markup
